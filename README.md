# CanvasHub

CanvasHub provides ready-to-use animated canvas backgrounds for the web, designed to be easy to explore, customize, and integrate.

## Features

- **Gallery** – browse animated HTML5 canvas backgrounds in a responsive grid
- **Live Preview** – see each background render in real time
- **Configurable Parameters** – tweak colors, speed, and other settings with instant feedback
- **Code Export** – copy self-contained HTML you can drop into any project

## Backgrounds

| Name | Description |
|------|-------------|
| Particle Field | Floating particles that drift and bounce around the canvas |
| Animated Waves | Smooth sine waves flowing across the canvas |
| Animated Gradient | Smoothly shifting color gradients that cycle through hues |

## Tech Stack

- [Vite](https://vite.dev/) + [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [React Router v7](https://reactrouter.com/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```
